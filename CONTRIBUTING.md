# Contributing to Vault to Env

Thanks for your interest in contributing.

## How to contribute

- **Bug reports and feature ideas:** Open an [issue](https://github.com/sandeepshekhawat/vault-to-env/issues).
- **Code changes:** Open a pull request. For larger changes, open an issue first to discuss.

## Development setup

1. Clone the repo and open the project in Xcode:
   ```bash
   git clone https://github.com/sandeepshekhawat/vault-to-env.git
   cd vault-to-env
   open VaultToEnvApp.xcodeproj
   ```
2. Let Xcode resolve the Yams package dependency.
3. Build (Cmd+B) and run (Cmd+R).
4. Run tests: **Product → Test** (Cmd+U).

## Code style

- Follow the existing Swift style in the project.
- Keep the app compatible with macOS 13+.

## Pull requests

- Keep PRs focused (one feature or fix per PR when possible).
- Ensure tests pass before submitting.
